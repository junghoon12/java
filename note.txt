/*
  반복 제어문
  - 특정 작업을 반복하여 수행하고자 할 때 사용하는 제어문.
  - 반복 제어문의 종류
    1) 기본 loop
       - 가장 기본적인 반복문.
    2) while loop
       - 특정 조건의 결과를 통해서 반복을 수행하는 반복문.
    3) for loop
       - 반복 횟수를 정하여 반복을 수행하는 반복문.
  - 반복문의 반복적인 수행을 종료시키는 명령어.
    1) exit
       - 수행 중인 반복을 종료시키는 명령어.
    2) exit ~ then
       - 반복 종료를 위한 조건식을 지정하고 만족하면
         반복을 종료시키는 명령어.
    3) continue
       - 수행 중인 반복의 현재 주기를 건너 뛰는 명령어.
    4) continue ~ when
       - 특정 조건식을 지정하고 조건식을 만족하면 반복 주기를
         건너 뛰는 명령어.
*/
