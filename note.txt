# utf-8-sig는 UTF-8에 **BOM(Byte Order Mark)**을 추가하여 Excel에서 한글을 올바르게 표시하게 도와줍니다.
# Excel에서 직접 열어도 한글이 깨지지 않음.
# **BOM(Byte Order Mark)**은 텍스트 파일의 인코딩 방식과 바이트 순서를 나타내는 특수한 숨겨진 문자입니다.
# UTF-8, UTF-16, UTF-32 같은 유니코드 인코딩에서 사용됨.
# 파일의 맨 앞에 추가되어 인코딩 방식을 식별하는 역할을 함.
# 일반적으로 눈에 보이지 않지만, 특정 프로그램(예: Excel, Notepad++)에서 인코딩을 판별하는 데 사용됨.
# UTF-8과 같은 유니코드 인코딩을 사용할 때, 파일을 읽는 프로그램이 올바른 인코딩을 자동으로 감지하도록 하기 위해 사용됨.
# 특히, Windows 메모장(Notepad)이나 Excel 같은 프로그램은 기본적으로 UTF-8 인코딩을 잘 감지하지 못하는 경우가 많음.
